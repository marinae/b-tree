# Структура репозитория:
```
classes.h     // Описание используемых структур

blocks.h      // Работа с блоками (страницами файла)
blocks.c

helpful.h     // Разные полезные функции
helpful.c

insert.h      // Вставка ключа в дерево
insert.c

select.h      // Поиск ключа в дереве
select.c

database.h    // Остальные (незаконченные) функции для работы с базой
database.c

mydb.h        // Инициализация базы данных и заполнение API
mydb.c
```
